The entire deploy surface, folded into your editor

The README is full of practical product detail, so this page keeps that fidelity: ShipShape is not just a shiny launch button. It is detection, config generation, logs, service management, and recovery, all designed around the VS Code workflow.

01

Smart project detection

Recognizes Next.js, Vite, Express, FastAPI, Django, Rails, Dockerfile setups, and generic Node or Python projects.

02

Auto `.locusbuild` generation

Writes a framework-appropriate config for you and shows a diff preview before you commit to the deploy.

03

Dockerfile rescue when needed

Corrects cases where default build behavior would bind to the wrong port, then commits and pushes the fix before deploy.

04

Service explorer sidebar

Browse Projects, Environments, Services, and Deployments with live status indicators and right-click actions.

05

Environment variable manager

Edit variables in a webview panel, save, and automatically trigger the redeploy required for them to take effect.

06

AI diagnosis and auto-fix loop

Gemini reads the failure logs and relevant files, explains what broke, and offers an apply-and-redeploy path when it is safe.

A deploy flow that feels like a feature, not a detour

ShipShape compresses the first-deploy ritual into a short guided loop. The goal is simple: keep developers in the editor and keep the cognitive load low.

1

Open the project

ShipShape checks for your Locus credentials, confirms billing readiness, and detects the framework automatically.

2

Review the config

If a `.locusbuild` file is missing, the extension generates one and shows it back to you before deployment starts.

3

Deploy from GitHub

Once the repo is confirmed, ShipShape triggers the Locus project-from-repo flow and opens a dedicated log channel.

4

Watch it go live

The status bar tracks progress from building to healthy, then surfaces the final HTTPS URL when the service is ready.

Install and ship

1. Install the packaged extension with code --install-extension shipshape-0.1.0.vsix.
2. Run ShipShape: Configure Locus API Key and paste your claw_... key.
3. Optionally add a Gemini key to unlock AI failure diagnosis and the auto-fix loop.
4. Open a GitHub-backed app in VS Code and trigger ShipShape: Deploy Workspace.

Security and trust

Encrypted storage. API keys live in VS Code SecretStorage, not in plain settings.
Short-lived auth. JWT tokens are cached in memory only and refreshed when needed.
Focused integrations. Secrets are sent only to Locus and Google for the AI diagnosis path.
Clear recovery path. When deploys fail, the extension surfaces logs, rollback options, and guided fixes instead of dead ends.

Turn VS Code into your Locus launch deck
